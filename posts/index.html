Spatial Vessels Analysis

Spatial Vessel Analysis Overview The following pipeline offers a robust, reproducible, and scalable solution for extracting vessel-related features from histological images. Accurate vascular network segmentation provides key insights into vascular conformation and remodeling. Additionally, this pipeline correlates vascular structures with drug distribution. While particularly relevant in oncology for characterizing the tumor microenvironment, it may also be applied to other diseases where vascular alterations are paramount, including those involving inflammation, fibrosis, and microvascular dysfunction. By enhancing our understanding of vascular-mediated disease progression and treatment delivery, this approach supports both research and clinical decision-making. Objectives Quantitative Analysis: Extract precise vessel morphology and spatial distribution metrics. Automated &amp; Scalable: Process large datasets efficiently, reducing manual workload. Open-Source &amp; Modifiable: Easily adaptable to different research contexts and imaging modalities. Interdisciplinary Integration: Combines pathology and bioinformatic image analysis. Getting Started 1. Prepare Your Data 1.1 Stain the Histological Images of Interest for CD31 Obtain histological images that include a stain to highlight blood vessels. Common choices include CD31 or CD34. CD31 is particularly useful due to its strong and specific endothelial staining, while CD34 provides additional coverage of endothelial and progenitor cells. These stains are essential for enhancing vessel contrast and enabling further analyses. Finally, ensure that the slides are digitally scanned for subsequent evaluation. 1.2 Ensure the Images Are Formatted Correctly Make sure the images are in a compatible format, such as .tiff (higher quality, slower processing) or .jpg (lower quality, faster processing). 2. Segment Vessels and Extract Features in Python With your images prepared, you can perform vessel segmentation and feature extraction using Python. The following steps outline the process. 2.0 Preliminary Setup Upload the necessary libraries: import cv2 import numpy as np import matplotlib.pyplot as plt from skimage.measure import label, regionprops from skimage import measure import pandas as pd import os Define the path to the image folders and create a list of subdirectories: PATH = &#34;Insert/Path/to/Folders&#34; folders = [item for item in os.listdir(PATH) if os.path.isdir(os.path.join(PATH, item))] Loop through each folder to process the images, construct the image path, and read the image using OpenCV. Create a directory for saving segmented images if it does not already exist: for folder in folders: general_path = os.path.join(PATH, folder) image_path = general_path + &#34;/&#34; + folder + &#34;.jpg&#34; original = cv2.imread(image_path) print(f&#34;Image - {image_path} is read&#34;) if not os.path.exists(f&#34;{general_path}/{folder}_SEGMENTATION&#34;): os.makedirs(f&#34;{general_path}/{folder}_SEGMENTATION&#34;) directory_to_save = general_path + &#34;/&#34; + folder + &#34;_SEGMENTATION/&#34; 2.1 Conversion of RGB to HSV Format Convert the images from the standard RGB (Red, Green, Blue) colour space to HSV (Hue, Saturation, Value) format. This conversion enhances the contrast of blood vessels: h_s_v_image = cv2.cvtColor(original, cv2.COLOR_BGR2HSV) h, s, v = cv2.split(h_s_v_image) h = 255 - h 2.2 Application of Otsu’s Thresholding Apply Otsu’s thresholding on the H channel to create a binary mask that separates vessel structures from the background: ret, binary = cv2.threshold(h, 0, 255, cv2.THRESH_OTSU) binary = binary.astype(np.uint8) binary *= 255 2.3 Morphological Operations Perform morphological operations to clean up noise and ensure vessel structures are continuous. Adjust the parameters as needed: kernel_for_structuring_element_for_segmentation = (15, 15) kernel_for_structuring_element_for_closing = (15, 15) iterations_for_dilate = 4 iterations_for_closing = 3 # Dilation to bridge gaps kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, kernel_for_structuring_element_for_segmentation) binary = cv2.dilate(binary, kernel, iterations=iterations_for_dilate) cnts = cv2.findContours(binary, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE) cnts = cnts[0] if len(cnts) == 2 else cnts[1] for c in cnts: cv2.drawContours(binary, [c], 0, (255, 255, 255), -1) # Closing to fill small gaps in vessels kernel = cv2.getStructuringElement(cv2.MORPH_RECT, kernel_for_structuring_element_for_closing) opening = cv2.morphologyEx(binary, cv2.MORPH_OPEN, kernel, iterations=iterations_for_closing) # Save the segmented binary image plt.imsave(f&#34;{directory_to_save}{folder}_Segmented.png&#34;, opening, cmap=&#34;gray&#34;) Example output: A binary image where vessels are white and the background is black. Overlay vessel contours on the original image for enhanced interpretability: contours, hierarchy = cv2.findContours(opening, cv2.RETR_TREE, cv2.CHAIN_APPROX_SIMPLE) for c in contours: cv2.drawContours(original, [c], 0, (0, 255, 0), 5) cv2.imwrite(f&#34;{directory_to_save}{folder}_Vessels.jpg&#34;, original) Example output: The original histological image with vessel contours highlighted in green. 2.4 Feature Extraction Extract quantitative features of vessel morphology using regionprops_table from skimage.measure. The features include: area: The number of pixels inside each segmented vessel. axis_major_length: The length of the vessel’s major axis. axis_minor_length: The length of the vessel’s minor axis. eccentricity: How close the vessel shape is to a circle (0 = perfect circle, 1 = elongated). orientation: The angle of the major axis relative to the horizontal axis. Label the segmented vessels and extract the desired properties: label_img = label(opening) all_props = [&#34;area&#34;, &#34;axis_major_length&#34;, &#34;axis_minor_length&#34;, &#34;eccentricity&#34;, &#34;orientation&#34;] props = measure.regionprops_table(label_img, original, properties=all_props) data = pd.DataFrame(props) data.to_csv(f&#39;{directory_to_save}Measurements_{folder}.csv&#39;, index=True) The output is a CSV file ({folder}_Measurements.csv) containing the quantitative measurements for each detected vessel. 3. Infer Drug Distributions This section explains how to correlate vascular network morphology with MALDI imaging data using Gaussian Process (GP) Modelling. The goal is to model the spatial distribution of a drug within a tissue sample based on MALDI imaging, where pixel intensity represents drug concentration. Before applying GP modelling to real data, the model is validated using simulations. Theoretical Background A Gaussian Process (GP) is a collection of random variables, any finite subset of which follows a multivariate normal distribution. It provides a probabilistic framework for modelling spatial dependencies and is widely used in spatial statistics and machine learning. 3.1 Generating Synthetic Vessel Images Execute the Final_optimized_4_Windows.R script (Windows only). Begin by setting up the environment, including setting a fixed seed and importing necessary libraries: set.seed(20241028) library(rethinking) library(parallel) library(plot.matrix) library(viridis) library(MASS) num_cores &lt;- detectCores() - 1 cl &lt;- makeCluster(num_cores) Simulate binary vessel masks as square matrices, where 1 represents a vessel and 0 represents non-vascular tissue: test_function &lt;- function(n, circles) { mat &lt;- matrix(0, nrow = n, ncol = n) for(i in 1:n) { for(j in 1:n) { for(circle in circles) { center &lt;- circle$center radius &lt;- circle$radius if (sqrt((i - center[1])^2 + (j - center[2])^2) &lt;= radius) { mat[i, j] &lt;- 1 } } } } return(mat) } Generate 10 masks, each with a 20x20 grid containing randomly sampled vessel positions and radii: N_img &lt;- 10 # Number of synthetic images n &lt;- 20 # Matrix dimensions circles_list &lt;- vector(&#34;list&#34;, N_img) for(i in 1:N_img) { circles_list[[i]] &lt;- list( list(center = c(sample(5:15, 1), sample(5:15, 1)), radius = runif(1, 1, 3)), list(center = c(sample(5:15, 1), sample(5:15, 1)), radius = runif(1, 1, 3)) ) } Generate the vessel masks in parallel: clusterExport(cl, c(&#34;test_function&#34;, &#34;n&#34;, &#34;circles_list&#34;)) mats &lt;- parLapply(cl, 1:N_img, function(i) test_function(n, circles_list[[i]])) 3.2 Creating Simulated MALDI Images Generate simulated MALDI intensity maps using an exponential decay model: [ I(x) = \eta^2 , e^{-\frac{d(x)^2}{2\rho^2}} ] Here, ( d(x) ) is the Euclidean distance from vessels, ( \eta^2 ) is the signal variance, and ( \rho ) is the characteristic length scale. Compute Euclidean distance matrices for each image: grids &lt;- parLapply(cl, 1:N_img, function(i) expand.grid(X = 1:n, Y = 1:n)) Dmats &lt;- parLapply(cl, 1:N_img, function(i) as.matrix(dist(grids[[i]], method = &#34;euclidean&#34;))) Define kernel function parameters: beta &lt;- 5 etasq &lt;- 2 rho &lt;- sqrt(0.5) Compute covariance matrices using the radial basis function (RBF) kernel: clusterExport(cl, c(&#34;beta&#34;, &#34;etasq&#34;, &#34;rho&#34;, &#34;Dmats&#34;)) Ks &lt;- parLapply(cl, 1:N_img, function(i) { etasq * exp(-0.5 * ((Dmats[[i]] / rho)^2)) + diag(1e-9, n*n) }) Sample synthetic MALDI intensities using a GP prior: clusterExport(cl, &#34;Ks&#34;) sim_gp &lt;- parLapply(cl, 1:N_img, function(i) { MASS::mvrnorm(1, mu = rep(0, n*n), Sigma = Ks[[i]]) }) Generate observed intensity values by adding noise: clusterExport(cl, c(&#34;sim_gp&#34;)) sim_y &lt;- parLapply(cl, 1:N_img, function(i) { rnorm(n*n, mean = sim_gp[[i]] + beta * as.vector(t(mats[[i]])), sd = 1) }) 3.3 Inferring Drug Distribution with GP Regression Fit a Gaussian Process model using Bayesian inference with Stan (via the rethinking package). Prepare the data for the model: dat_list &lt;- list(N = n*n) for(i in 1:N_img) { dat_list[[ paste0(&#34;y&#34;, i) ]] &lt;- sim_y[[i]] dat_list[[ paste0(&#34;x&#34;, i) ]] &lt;- as.vector(t(mats[[i]])) dat_list[[ paste0(&#34;Dmat&#34;, i) ]] &lt;- Dmats[[i]] } Define the GP model: model_code &lt;- &#34;alist(\n&#34; for(i in 1:N_img) { model_code &lt;- paste0(model_code, &#34; y&#34;, i, &#34; ~ multi_normal(mu&#34;, i, &#34;, K&#34;, i, &#34;),\n&#34;, &#34; mu&#34;, i, &#34; &lt;- a + b * x&#34;, i, &#34;,\n&#34;, &#34; matrix[N, N]:K&#34;, i, &#34; &lt;- etasq * exp(-0.5 * square(Dmat&#34;, i, &#34; / rho)) + diag_matrix(rep_vector(0.01, N)),\n&#34;) } model_code &lt;- paste0(model_code, &#34; a ~ normal(0, 1),\n&#34;, &#34; b ~ normal(0, 0.5),\n&#34;, &#34; etasq ~ exponential(2),\n&#34;, &#34; rho ~ exponential(0.5)\n&#34;, &#34;)&#34;) model_list &lt;- eval(parse(text = model_code)) Fit the GP model using Hamiltonian Monte Carlo (HMC): GP_N &lt;- ulam(model_list, data = dat_list, chains = 4, cores = num_cores, iter = 600, warmup = 150) Print the model summary: print(precis(GP_N)) Extract posterior samples: post &lt;- extract.samples(GP_N) 3.4 Validating the Model Visualize the inferred vs. true covariance functions by plotting the priors, the actual kernel, and the estimated kernels: set.seed(08062002) p.etasq &lt;- rexp(n, rate = 0.5) p.rhosq &lt;- rexp(n, rate = 0.5) plot(NULL, xlim = c(0, max(Dmats[[1]])/3), ylim = c(0, 10), xlab = &#34;pixel distance&#34;, ylab = &#34;covariance&#34;, main = &#34;Prior, Actual, and Estimated Kernel&#34;) # Priors for(i in 1:20) curve(p.etasq[i] * exp(-0.5 * (x/p.rhosq[i])^2), add = TRUE, lwd = 6, col = col.alpha(2, 0.5)) # Actual kernel curve(etasq * exp(-0.5 * (x/rho)^2), add = TRUE, lwd = 4) # Estimated kernels for(i in 1:20) { curve(post$etasq[i] * exp(-0.5 * (x/post$rho[i])^2), add = TRUE, col = col.alpha(4, 0.3), lwd = 6) } stopCluster(cl) If the inferred decay function closely matches the predefined function, the GP model has accurately recovered the known distribution parameters and is ready to be applied to real-world histological and MALDI images. 4. Interpret Your Data This final step involves translating the extracted quantitative measurements into biological insights. Consider the implications of vessel morphology and drug distribution in the context of disease progression and treatment efficacy. The resulting data can guide future research directions and inform clinical decision-making. Contributors G. Grion, K. Roufail, F. E. Colella, Dr. Ö. Mintemur, Dr. S. L. Renne.
